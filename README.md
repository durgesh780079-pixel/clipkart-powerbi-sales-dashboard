# ClipKart Sales Performance Dashboard

## Overview
This project is an interactive Power BI dashboard created to analyze sales performance for ClipKart.

The dashboard provides insights into:
- Total Revenue
- Total Orders
- Quantity Sold
- Average Order Value
- Top Revenue Products
- Top Performing Cities
- Category-wise Revenue Distribution
- Revenue Trends by Price

## Tools Used
- Power BI
- Excel / CSV Dataset
- Data Visualization

## Dashboard Features
- KPI Cards for key business metrics
- Revenue analysis by category
- Top performing products
- City-wise sales performance
- Interactive charts and visuals
- Business performance tracking

## Key Insights
- Total Revenue: 6.157M
- Total Orders: 1K
- Quantity Sold: 2K
- Average Order Value: 6.16K

## Project Goal
To help businesses make data-driven decisions by visualizing sales performance and identifying growth opportunities.
